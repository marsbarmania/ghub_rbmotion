## Rubymotion with Cocoapods Sample

####インストールとセットアップ

<code>% gem install cocoapods</code><br>
<code>% pod setup</code><br>
Rubymotion用のgemをインストールする。<br>
<code>% gem install motion-cocoapods</code><br>
